# FAKE-NEWS-DETECTION-AI-ML-
Fake News Detection System (Machine Learning Project)

This project presents a comprehensive Fake News Detection System developed using advanced Machine Learning and Deep Learning techniques to classify news articles as real or fake. The system leverages natural language processing (NLP) methods combined with multiple predictive models to ensure accurate and reliable classification.

Project Overview

With the rapid spread of misinformation across digital platforms, detecting fake news has become a critical challenge. This project aims to address this issue by building a robust classification pipeline that analyzes textual data and predicts the authenticity of news content.

The system incorporates data preprocessing, feature extraction, model training, evaluation, and visualization, providing an end-to-end solution for fake news detection.

Technologies & Techniques Used

Natural Language Processing (NLP):
Text cleaning (removal of stopwords, punctuation, etc.)
Tokenization and normalization
Feature extraction using TF-IDF (Term Frequency–Inverse Document Frequency)

Machine Learning Algorithms:
Logistic Regression – for binary classification
K-Nearest Neighbors (KNN) – for similarity-based classification
Linear Regression (Approximation) – for exploratory prediction analysis
K-Means Clustering – for unsupervised pattern grouping

Deep Learning:
Neural Networks (implemented using Keras) for improved prediction accuracy on complex patterns

Data Visualization:
Implemented using matplotlib for insightful graphical analysis

Project Structure

fake_news_final_project.py
Contains the complete implementation including preprocessing, model training, evaluation, and visualization.

Fake.csv
Dataset containing fake news articles.

True.csv
Dataset containing genuine news articles.

Models Implemented

The project evaluates and compares multiple models to determine the most effective approach:

Logistic Regression
K-Nearest Neighbors (KNN)
Linear Regression (for approximation purposes)
K-Means Clustering
Neural Networks (Keras-based implementation)

Data Visualization

To better understand the dataset and model performance, the following visualizations are included:

Histograms – distribution of features
Bar Graphs – categorical comparisons
Line Charts – trend analysis
Scatter Plots – relationship between variables
Pie Charts – class distribution
